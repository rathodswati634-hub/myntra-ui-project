# 🛍️ Myntra Inspired Shopping UI

## 📊 Case Study

This project is a simple Mantra Name Generator inspired by filtering and sorting systems.

## 🎯 Objective
- Implement filtering and sorting logic
- Improve UI interaction
- Understand dynamic UI behavior

## 🔍 Features
- Category Filtering
- Length Filtering
- Sorting (A-Z, Length)
- Real-time updates

## ⚙️ Working
- Data stored in JavaScript array
- Filtering using .filter()
- Sorting using .sort()
- Dynamic DOM updates

## 🛠 Technologies
- HTML5
- CSS3
- JavaScript
